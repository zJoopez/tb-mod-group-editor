This is an editor for Toribash mods<br>
Goal is to provide good interface for modifying multiple objects at once as the default editor only allows adjusting one object at a time<br>

Installation<br>
You can grab latest version from releases and extract that to script folder of the game<br>

How to use<br>
you can launch mge.lua via ingame script interface or use /ls mge.lua once you've added files to script folder
